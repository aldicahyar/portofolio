# Resume File

Place your resume PDF file here with the filename: `resume.pdf`

The download button in the Hero section will automatically download this file as `Aldi_Cahya_Ramadhan_Resume.pdf`.

## File Requirements

- **Filename:** `resume.pdf` (exact name required)
- **Format:** PDF
- **Location:** `/public/resume.pdf`
- **Recommended size:** < 5MB

## Tips for Your Resume

1. Keep it to 1-2 pages
2. Use a clean, professional format
3. Include:
   - Contact information
   - Professional summary
   - Work experience
   - Education
   - Technical skills
   - Projects (if relevant)
4. Export as PDF to preserve formatting
